## 2. エネルギー代謝の基礎生化学（復習）

### 2.1 脂肪酸酸化の律速段階

```
脂肪組織 → FFA放出（HSL/ATGL）
    ↓
血中アルブミン結合で運搬
    ↓
筋細胞へ取り込み
    ↓
Acyl-CoA Synthetase → Acyl-CoA
    ↓
★ CPT-1（律速酵素）→ ミトコンドリアへ輸送
    ↓
β酸化 → アセチルCoA → TCAサイクル → ATP
```

**CPT-1は Malonyl-CoA により阻害される。糖質摂取→インスリン→ACC活性化→Malonyl-CoA↑→CPT-1ブロック→脂肪燃焼停止**

### 2.2 Randle Cycle の核心

> グルコースと脂肪酸は細胞内で基質として競合する。糖質を多く摂ると脂肪酸酸化がブロックされ、脂肪燃焼が止まる。

### 2.3 AMPK軸

```
エネルギー不足（運動・空腹）→ AMPK活性化
    ↓
ACC不活性化 → Malonyl-CoA↓ → CPT-1解放 → 脂肪燃焼
    ↓（同時に）
PGC-1α発現 → ミトコンドリア新生 → 脂肪燃焼能力の器が拡大
```

## 3. 体脂肪率20〜30%帯の代謝特性（詳細）

### 3.1 主要な代謝障害

- **代償性高インスリン血症**: 同じ糖質量でもインスリンが過剰に分泌される
- **カテコラミン抵抗性**: β-AR機能低下により運動時の脂肪分解応答が鈍い
- **レプチン抵抗性**: 食欲抑制シグナルが機能不全
- **アディポネクチン低下**: AMPK経路の活性が弱い
- **慢性微小炎症**: TNF-α、IL-6による全身性インスリン抵抗性
- **異所性脂肪蓄積**: 肝臓・筋肉へのDAG/セラミド蓄積

### 3.2 なぜ低体脂肪の人と同じ食事では脂肪が落ちないか

```
【体脂肪率25%】同じ糖質60g摂取
 → インスリン分泌: 大量（抵抗性により代償的過剰分泌）
 → Malonyl-CoA: 大量生成
 → CPT-1: 強力にブロック
 → 結果: 脂肪燃焼が長時間停止

【体脂肪率12%】同じ糖質60g摂取
 → インスリン分泌: 適量（高感受性）
 → Malonyl-CoA: 限定的
 → CPT-1: 軽度の一時的阻害のみ
 → 結果: 速やかに脂肪燃焼に復帰
```

## 4. 体脂肪率10〜20%帯の代謝特性（新規）

### 4.1 15〜20%帯：「代謝回復のスイートスポット」

この帯域は代謝環境が大きく改善し始めるゾーン：

- **インスリン感受性**: 顕著に改善。糖質への耐性が回復する
- **代謝柔軟性**: 空腹時の脂肪酸酸化と食後のグルコース酸化の切り替えがスムーズになる
- **ホルモン環境**: テストステロンが最適域に近づく。レプチンが適正化し食欲管理が容易に
- **運動パフォーマンス**: 糖質の活用効率が上がり、高強度トレーニングのパフォーマンスが向上
- **IMTGターンオーバー**: 運動習慣と相まって改善し始める

### 4.2 10〜15%帯：「パフォーマンスと代償のバランス」

この帯域では新たなリスクが出現する：

| 指標 | 15%時点 | 10%時点 |
|---|---|---|
| テストステロン | 健常範囲 | **30〜75%低下のリスク** |
| コルチゾール | 正常〜やや上昇 | **慢性的上昇** |
| T3（甲状腺ホルモン） | 正常 | **低下（代謝適応）** |
| レプチン | 適正 | **著しく低下→強い空腹感** |
| GH | 最適 | **パルス減少リスク** |
| 基礎代謝率 | ほぼ正常 | **適応性熱産生で15〜25%低下** |
| NEAT | 維持可能 | **無意識に300〜500kcal/日低下** |
| 免疫機能 | 正常 | **上気道感染リスク上昇** |
| 筋タンパク質合成(MPS) | 正常レスポンス | **アナボリック抵抗性出現** |

### 4.3 T/C比（テストステロン/コルチゾール比）の重要性

```
T/C比 > 0.50   → アナボリック優位（筋肉維持・成長に有利）
T/C比 0.30-0.50 → バランス域（適切な管理で維持可能）
T/C比 < 0.30   → カタボリック優位（筋分解リスク大・要注意）
```

体脂肪率10%以下への到達は、T/C比を急激に悪化させる可能性があり、**コンテスト目的以外では推奨されない**。

### 4.4 「アスリートのパラドックス」と10〜15%帯

体脂肪率が15%以下になると、IMTG（筋肉内トリグリセリド）の管理が変化する：

- アスリートはIMTGを「積極的に使い、積極的に補充する」高ターンオーバー状態
- これには十分なミトコンドリア密度と酸化能力が前提
- この帯域では、適度な脂質摂取がホルモン維持とIMTGターンオーバーに必須

## 6. 総エネルギー消費量と適応性熱産生

### 6.1 TDEEの構成と各フェーズでの変化

```
TDEE = BMR(60-70%) + TEF(10%) + EAT(5-15%) + NEAT(15-30%)
```

| 栄養素 | TEF（食事誘発性熱産生） |
|---|---|
| タンパク質 | **20〜30%** |
| 炭水化物 | 5〜10% |
| 脂質 | 0〜3% |

### 6.2 適応性熱産生への対策

減量が進むと体は代謝を下げて抵抗する。対策：

1. **NEAT維持**: 1日10,000歩以上を意識的に維持
2. **リフィード**: 2〜3週に1日、糖質を増やしてレプチン・T3を一時回復
3. **ダイエットブレイク**: 8〜12週ごとに1〜2週間のメンテナンスカロリー
4. **タンパク質優先**: TEFが最も高い栄養素で代謝低下を抑制

1. **カロリー赤字は必要条件だが、マクロ構成が除脂肪の効率を決定する**
